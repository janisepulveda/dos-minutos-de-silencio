# dos-minutos-de-silencio

El objetivo de este encargo es presentar en dos minutos, una reflexión personal sobre el impacto que tuvo el curso en ustedes, usando sólo 1 lámina/diapositiva.

NO es un resumen del curso NI una evaluación del curso o del profesor. Lo que busca este encargo es que nos presenten que cambio (transformación) se produjo en ustedes como resultado de haber tomado el curso, más allá de los contenidos aprendidos y aplicados: este cambio puede ser en la forma de ver / entender los servicio, el diseño, la experiencia, etc... o puede relacionarse con la interdisciplina, o la mirada/aprendizaje que tienen de algún tema en especifico, o lo que se les ocurra que es relevante y es consecuencia del curso. Lo que se busca es una reflexion personal, donde no hay respuestas correctas o incorrectas.
